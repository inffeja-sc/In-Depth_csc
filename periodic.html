🔄 Перевод периодических P-ичных дробей

Существует несколько способов перевода бесконечной периодической дроби в десятичную систему. Все они касаются лишь преобразования периода, а непериодическую дробную часть следует переводить отдельно.

Способ 1: Умножение на p^k

Пусть дробь имеет вид a = 0,(a₁a₂...aₖ)_p — чисто периодическая дробь. Умножим эту дробь на p^k, т.е. сдвинем запись на k позиций вправо:

b = a · pᵏ = a₁a₂...aₖ,(a₁a₂...aₖ)_p = a₁a₂...aₖ + a

Отсюда получаем уравнение:

a · pᵏ = a + a₁a₂...aₖ a = (a₁a₂...aₖ) / (pᵏ - 1)

Пример 24. Перевод 0,(1001)₂

b = 2⁴ · a = 1001,(1001)₂ = a + 1001₂ 16 · a = a + 9 15 · a = 9 a = 9/15 = 0,6₁₀

При наличии непериодической части вычисления изменяются незначительно. Пусть периодическая P-ичная дробь имеет вид:

b = 0,c₁c₂...cₙ(a₁a₂...aₖ)_p

Обозначим через a дробь a = 0,(a₁a₂...aₖ)_p. Тогда:

b = a · p⁻ⁿ + 0,c₁c₂...cₙ

Пример 25. Перевод 0,10(1001)₂

0,10(1001)₂ = 2⁻² · 0,(1001)₂ + 0,1₂ = 0,25₁₀ · 0,6₁₀ + 0,5₁₀ = 0,15₁₀ + 0,5₁₀ = 0,65₁₀ 0,00(1001)₂ = 2⁻² · 0,(1001)₂ = 0,25₁₀ · 0,6₁₀ = 0,15₁₀

Непериодическая часть последней двоичной дроби состоит из двух нулей, но её нельзя считать отсутствующей, так как она определяет сдвиг периода.

Способ 2: Бесконечная геометрическая прогрессия

Запишем исходную бесконечную периодическую дробь в виде суммы:

0,c₁c₂...cₙ(a₁a₂...aₖ)_p = 0,c₁c₂...cₙ + p⁻ⁿ·a₁a₂...aₖ + pⁿ⁻ᵏ·a₁a₂...aₖ + ...

За исключением непериодической части данное выражение соответствует сумме бесконечной геометрической прогрессии со знаменателем q = p⁻ᵏ < 1 и первым членом p⁻ⁿ · a₁a₂...aₖ:

S = (p⁻ⁿ · a₁a₂...aₖ) / (1 - p⁻ᵏ) = (a₁a₂...aₖ) / (pⁿ · (pᵏ - 1))

Пример 26. Перевод 0,10(1001)₂ через прогрессию

0,10(1001)₂ = 0,1₂ + 2⁻²·1001₂ + 2⁻⁶·1001₂ + 2⁻¹⁰·1001₂ + ... Сумма прогрессии: (2⁻² · 1001₂) / (1 - 2⁻⁴) = (0,25 · 9) / (1 - 0,0625) = 2,25 / 0,9375 = 0,15 Прибавляем непериодическую часть: 0,1₂ = 0,5₁₀ Итого: 0,15 + 0,5 = 0,65₁₀
