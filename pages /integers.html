🔢 Перевод целых чисел

Из десятичной в систему с основанием p

Развернутую запись числа можно использовать для обратного перевода — из десятичной системы в систему с основанием p. Действительно, из формулы:

a₃a₂a₁a₀ = a₃·p³ + a₂·p² + a₁·p¹ + a₀

следует, что a₀ — это остаток от деления исходного числа на основание p. Если мы разделим исходное число на p и отбросим остаток, мы получим:

a₃a₂a₁ = a₃·p² + a₂·p¹ + a₁

Теперь легко найти a₁ — это последняя цифра получившегося числа, которая равна остатку от его деления на p.

📌 Алгоритм перевода

Для перевода числа из десятичной системы в систему счисления с основанием p нужно:

  1. Делить число на p, отбрасывая остаток на каждом шаге, пока не получится 0
  2. Выписать полученные остатки в обратном порядке

Пример. Перевод 194 в пятеричную систему (p=5)

Найдём остаток от деления на 5:

194 = 38·5 + 4 → последняя цифра: 4 38 = 7·5 + 3 → следующая цифра: 3 7 = 1·5 + 2 → следующая цифра: 2 1 = 0·5 + 1 → первая цифра: 1

Полученные остатки нужно выписать в обратном порядке:

194₁₀ = 1234₅

⚠️ Важно

При p > 10 используются также и латинские буквы (A=10, B=11, C=12, D=13, E=14, F=15 и т.д.).

Из двоичной в десятичную

Для того чтобы перевести число из двоичной системы счисления в десятичную, надо в десятичной системе счисления сложить все степени двоек, которые соответствуют единицам в записи исходного двоичного числа.

Пример 19. Перевод 1001101₂ в десятичное

1001101₂ = 2⁰ + 2² + 2³ + 2⁶ = 1 + 4 + 8 + 64 = 77₁₀

🎯 Полезно знать

Первые шестнадцать степеней двойки:

2³ = 8 2⁷ = 128 2¹¹ = 2048 2¹⁵ = 32768 2⁴ = 16 2⁸ = 256 2¹² = 4096 2¹⁶ = 65536 2⁵ = 32 2⁹ = 512 2¹³ = 8192 2⁶ = 64 2¹⁰ = 1024 2¹⁴ = 16384

Нахождение основания системы

Пример 1. Линейное уравнение

Число 71 в некоторой системе с основанием x записывается как 56ₓ. Представим это число в развернутой форме:

71 = 56ₓ = 5·x¹ + 6·x⁰ = 5·x + 6

Решая уравнение 71 = 5·x + 6, получаем: x = 13

Пример 2. Квадратное уравнение

Число 71 в некоторой системе записывается как 155ₓ:

71 = 155ₓ = 1·x² + 5·x¹ + 5·x⁰ = x² + 5·x + 5

Решая уравнение 71 = x² + 5·x + 5, получаем два решения: x₁ = -11 и x₂ = 6. Искомое основание положительно, поэтому правильный ответ — 6.

Пример 3. Несколько решений

Найдем все основания систем счисления, в которых запись десятичного числа 24 оканчивается на 3. Используем схему Горнера:

24 = k·x + 3 → 21 = k·x

Все интересующие нас основания являются делителями числа 21: это могут быть 3, 7 и 21. Поскольку последняя цифра числа — 3, основание не может быть равно 3 (в троичной системе нет цифры 3), поэтому условию задачи удовлетворяют только основания 7 и 21.

Пример 4. Числа с заданным окончанием

Найдем все десятичные числа, не превосходящие 40, запись которых в системе с основанием 4 оканчивается на 11:

N = k·4² + 1·4 + 1 = k·16 + 5

Подставляя k = 0, 1, 2, 3, ..., находим: N = 5, 21, 37, 53, ... Из них только 5, 21 и 37 удовлетворяют условию (не больше 40).

Пример 5. Слова как числа

Все 5-буквенные слова, составленные из букв А, О, У, записаны в алфавитном порядке. Заменив буквы А, О, У соответственно на цифры 0, 1 и 2, получаем числа в троичной системе. На 140-м месте стоит десятичное число 139:

139 = 12011₃ → УАООО
