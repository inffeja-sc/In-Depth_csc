🧮 Дробные числа

Дробные числа сначала рассмотрим на примере десятичной системы. Число 0,6375 можно представить в виде:

0,6375 = 6·0,1 + 3·0,01 + 7·0,001 + 5·0,0001

Все множители, на которые умножаются значения цифр, представляют собой отрицательные степени числа 10. Можно использовать развернутую форму записи, вводя отрицательные разряды:

разряды → -1 -2 -3 -4 6 3 7 5 0,6375 = 6·10⁻¹ + 3·10⁻² + 7·10⁻³ + 5·10⁻⁴

Схема Горнера для дробного числа:

0,6375 = 10⁻¹ · (6 + 10⁻¹ · (3 + 10⁻¹ · (7 + 10⁻¹ · 5)))

Общий случай для основания p

Рассмотрим дробное число 0,a₁a₂a₃a₄, записанное в системе счисления с основанием p:

0,a₁a₂a₃a₄ = a₁·p⁻¹ + a₂·p⁻² + a₃·p⁻³ + a₄·p⁻⁴ или схема Горнера: 0,a₁a₂a₃a₄ = p⁻¹ · (a₁ + p⁻¹ · (a₂ + p⁻¹ · (a₃ + p⁻¹ · a₄)))

📌 Алгоритм перевода дробной части

Умножив это число на p, получаем a₁,a₂a₃a₄. Если взять целую часть результата, мы получим цифру a₁. Таким же способом можно найти остальные цифры дробной части:

  1. На каждом шаге умножаем дробную часть на p
  2. Запоминаем целую часть результата — это очередная цифра
  3. Продолжаем, пока дробная часть не станет равна нулю

Пример. Перевод 0,9376 в пятеричную систему

Вычисления Целая часть Дробная часть
0,9376 · 5 = 4,68840,688
0,688 · 5 = 3,4430,44
0,44 · 5 = 2,220,2
0,2 · 5 = 110

Выписываем все целые части результатов:

0,9376 = 0,4321₅

⚠️ Бесконечные дроби

Вычисления заканчиваются, когда при очередном умножении дробная часть результата равна нулю. Но не всегда это происходит! Например, число 0,3 в пятеричной системе даёт бесконечную дробь. В этом случае обычно задают нужное количество значащих цифр и округляют число соответствующим образом.

Перевод конечных P-ичных дробей в десятичную

Способ 1: Развернутая форма

Представим дробь в развернутой форме и вычислим значение многочлена в десятичной системе:

Пример 20. Перевод 0,B0F9₁₆ в десятичную

0,B0F9₁₆ = 11·16⁻¹ + 0·16⁻² + 15·16⁻³ + 9·16⁻⁴ = 0,6912994384765625₁₀

Способ 2: Обыкновенная дробь

Представим конечную P-ичную дробь в виде обыкновенной дроби. Числителем будет число после запятой, а знаменателем — p^k, где k — количество значащих цифр в дробной части.

Пример 22. Перевод 0,13₅ в десятичную

0,13₅ = 13₅ / 100₅ = 8₁₀ / 25₁₀ = 0,32₁₀

Пример 23. Перевод 0,1A₁₆ в десятичную

0,1A₁₆ = 1A₁₆ / 100₁₆ = 26₁₀ / 256₁₀ = 0,1015625₁₀

Схема Горнера для дробей

Пример 21. Вычисление 0,1101₂ по схеме Горнера

0,1101₂ = (((1·2⁻¹ + 1)·2⁻¹ + 0)·2⁻¹ + 1)·2⁻¹ = (((0,5 + 1)·0,5 + 0)·0,5 + 1)·0,5 = ((1,5·0,5 + 0)·0,5 + 1)·0,5 = (0,75·0,5 + 1)·0,5 = (0,375 + 1)·0,5 = 1,375·0,5 = 0,8125

Смешанные числа

Если нужно перевести число, в котором есть целая и дробная части, то части переводят отдельно, а потом соединяют.

Пример. Перевод 25,375 в пятеричную систему

25,375 = 25 + 0,375 25 = 100₅ 0,375 = 0,14₅ 25,375 = 100,14₅
