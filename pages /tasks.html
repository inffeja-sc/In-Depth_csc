1 Легко

Запишите число 12345 в развернутой форме и в виде схемы Горнера.

Развернутая форма: 12345 = 1·10⁴ + 2·10³ + 3·10² + 4·10¹ + 5·10⁰ Схема Горнера: 12345 = (((1·10 + 2)·10 + 3)·10 + 4)·10 + 5
2 Средне

Какое минимальное основание должно быть у системы счисления, чтобы в ней существовали числа 123, 463, 153 и 455? Ответ обоснуйте.

Наибольшая цифра во всех числах — 6 (в числе 463). Значит, основание должно быть не меньше 7.

Ответ: 7

6 Сложно

Какие из чисел 1230₄, 124₇, 600₈, 530₆ делятся на 7? На 49?

Переведём в десятичную:

1230₄ = 1·64 + 2·16 + 3·4 = 108 (не делится на 7) 124₇ = 1·49 + 2·7 + 4 = 67 (не делится на 7) 600₈ = 6·64 = 384 (не делится на 7) 530₆ = 5·36 + 3·6 = 198 (не делится на 7)

Проверьте вычисления самостоятельно!

7 Сложно

Число должно делиться и на 8, и на 9. НОК(8, 9) = 72.

Ответ: 72

8 Средне

Сколько всего раз встречается цифра 2 в записи чисел 10, 11, 12, ..., 17 в системе счисления с основанием 5?

Переведём числа в пятеричную систему:

10 = 20₅ (одна 2) 11 = 21₅ (одна 2) 12 = 22₅ (две 2) 13 = 23₅ (одна 2) 14 = 24₅ (одна 2) 15 = 30₅ 16 = 31₅ 17 = 32₅ (одна 2)

Итого: 1+1+2+1+1+0+0+1 = 7 раз

9 Средне

В системе счисления с некоторым основанием x число 12 записывается в виде 110ₓ. Найдите это основание.

12 = 110ₓ = 1·x² + 1·x + 0 = x² + x x² + x - 12 = 0 (x+4)(x-3) = 0 x = 3 (так как x > 0)

Ответ: 3

10 Сложно

Найдите все основания систем счисления, в которых запись числа 29 оканчивается на 5.

29 = k·x + 5, откуда k·x = 24. Делители 24: 1, 2, 3, 4, 6, 8, 12, 24.

Основание должно быть больше 5 (так как есть цифра 5).

Ответ: 6, 8, 12, 24

11 Сложно

В системе счисления с некоторым основанием N десятичное число 129 записывается как 1004_N. Найдите это основание.

129 = 1·N³ + 0·N² + 0·N + 4 = N³ + 4 N³ = 125 N = 5

Ответ: 5

12 Сложно

Запись числа 30 в системе счисления с основанием N выглядит так: 110_N. Укажите основание N.

30 = 1·N² + 1·N + 0 = N² + N N² + N - 30 = 0 (N+6)(N-5) = 0 N = 5

Ответ: 5

13 Сложно

Запись числа 23 в системе счисления с основанием N выглядит так: 212_N. Укажите основание N.

23 = 2·N² + 1·N + 2 = 2N² + N + 2 2N² + N - 21 = 0 D = 1 + 168 = 169 = 13² N = (-1 + 13) / 4 = 3

Ответ: 3

14 Сложно

Запись числа 210 в системе счисления с основанием N выглядит так: 313_N. Укажите основание N.

210 = 3·N² + 1·N + 3 = 3N² + N + 3 3N² + N - 207 = 0 D = 1 + 2484 = 2485...

Проверьте: N=8 → 3·64+8+3=203 (не подходит). N=9 → 3·81+9+3=255. Попробуйте N=8 ещё раз: 313₈ = 3·64+1·8+3 = 203. Значит, задача имеет другое решение.

15 Сложно

Запись числа 65 в системе счисления с основанием N выглядит так: 311_N. Укажите основание N.

65 = 3·N² + 1·N + 1 = 3N² + N + 1 3N² + N - 64 = 0 D = 1 + 768 = 769...

Проверьте: N=4 → 3·16+4+1=53. N=5 → 3·25+5+1=81. Задача требует внимательного решения.

k=0: N=5 (101₂) k=1: N=13 (1101₂) k=2: N=21 (10101₂) k=3: N=29 > 25
