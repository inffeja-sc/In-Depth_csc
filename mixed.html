🔀 Смешанные системы счисления

В некоторых случаях числа, заданные в системе счисления с основанием Q, приходится изображать с помощью цифр другой P-ичной системы счисления.

Определение 8

Системы счисления, в которых каждый коэффициент разложения числа по степеням Q (цифра Q-ичной системы счисления) записывается в P-ичной системе счисления, называются смешанными. Иначе такие системы называют P-Q-ичными.

Двоично-десятичная система

Ранее широкое распространение в вычислительной технике имела двоично-десятичная система. В ней основанием является число 10, но все десятичные цифры отдельно кодируются четырьмя двоичными цифрами и записываются последовательно друг за другом.

Так, число 8391 в двоично-десятичной системе будет записываться как 100000111001.

Заметим, что такое представление обладает избыточностью, поскольку четыре двоичные цифры могут кодировать не 10, а 16 различных чисел.

Случай Q = P^m

Особый интерес представляет случай, когда Q = P^m, где m — натуральное число. Для таких систем вид числа в P-Q-ичной системе совпадает с видом числа в P-ичной системе.

Теорема 2. Перевод из P в Q=P^m

Для того чтобы перевести целое число из системы счисления с основанием P в систему счисления с основанием Q=P^m, где m — натуральное число, достаточно запись числа в P-ичной системе разбить на группы по m цифр, начиная с правой цифры, и каждую такую группу заменить одной цифрой в Q-ичной системе.

Пример. Перевод 10101₂ в восьмеричную

10101₂ = 10|101 = 25₈ (P=2; Q=8; m=3)

Доказательство Теоремы 2

Запишем исходное число в развернутом виде в системах счисления с основаниями P и Q:

a₀ + a₁P + a₂P² + ... + aₙ₋₁Pⁿ⁻¹ = b₀ + b₁Q + b₂Q² + ... + bₙ₋₂Qⁿ⁻² (1.15)

Преобразуем выражение в левой части (разобьём на группы по m членов и заменим P^m на Q):

(a₀ + a₁P + ... + aₘ₋₁Pᵐ⁻¹) + Pᵐ(aₘ + aₘ₊₁P + ... + a₂ₘ₋₁Pᵐ⁻¹) + ... = Q⁰(a₀ + a₁P + ... + aₘ₋₁Pᵐ⁻¹) + Q¹(aₘ + aₘ₊₁P + ... + a₂ₘ₋₁Pᵐ⁻¹) + ...

Покажем, что любой многочлен в скобках строго меньше Q. Для этого каждую цифру P-ичной системы заменим на максимальную цифру [P-1]:

a₀ + a₁P + ... + aₘ₋₁Pᵐ⁻¹ ≤ (P-1)(1 + P + P² + ... + Pᵐ⁻¹) = (P-1)(Pᵐ-1)/(P-1) < Q

Следовательно, каждый многочлен в скобках при степенях Q можно записать в виде одной цифры Q-ичной системы. В силу единственности представления:

a₀ + a₁P + ... + aₘ₋₁Pᵐ⁻¹ = b₀ aₘ + aₘ₊₁P + ... + a₂ₘ₋₁Pᵐ⁻¹ = b₁ и т.д.

Теорема 3. Перевод из Q=P^m в P

Для того чтобы перевести целое число из системы счисления с основанием Q=P^m в систему счисления с основанием P, необходимо каждую Q-ичную цифру перевести в систему с основанием P и дополнить, если это необходимо, полученные числа слева нулями так, чтобы каждое число, за исключением самого левого, состояло ровно из m цифр.

Пример. Перевод 73₁₆ в двоичную

73₁₆ = 111|0011₂ = 1110011₂ (P=2; Q=16; m=4)

Практические применения

  1. Арифметические действия над числами, записанными в одной из таких систем, вы можете выполнять в другой системе, если последняя более удобна. Например, действия с шестнадцатеричными или восьмеричными числами легко заменяются на двоичную арифметику.
  2. Сокращение записи числа — замена системы счисления с меньшим основанием P на систему с большим основанием Q=P^m уменьшает количество цифр в m раз. Например, при использовании двоичной системы числа можно представлять в 16-ричной, сократив количество цифр в 4 раза (16=2⁴).
  3. Рациональный перевод — при переводе чисел из восьмеричной системы в шестнадцатеричную и наоборот удобно сначала переписать число в двоичном виде (8=2³, 16=2⁴).

Пример 32. Перевод 1010,00011011₂ в восьмеричную

Для двоичной и восьмеричной систем выполняется соотношение Q=P^m, а именно 8=2³. Группируем цифры по три:

1|010,|000|110|11, = 12,066₈

Последняя группа двоичных цифр была дополнена нулём справа.

Пример 33. Перевод A,1C₁₆ в четверичную

Для шестнадцатеричной и четверичной систем выполняется соотношение 16=4². Заменим каждую 16-ричную цифру её 4-ричным представлением:

A₁₆ = 10₁₀ = 22₄ C₁₆ = 12₁₀ = 30₄ A,1C₁₆ = 22,0130₄ = 22,013₄

Пример 34. Перевод BF3,6₁₆ в восьмеричную

BF3,6₁₆ = 1011|1111|0011,0110₂ = 101|111|110|011,011₂ = 5763,3₈
